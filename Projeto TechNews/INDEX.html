Notícia em Destaque

GitHub confirma uso de qualquer código público para o novo recurso Copilot, indiferente de licenças:

a inteligência artificial Codex, que é a base do Copilot, foi treinada em todos os repositórios públicos disponíveis no GitHub, sem distiguir o tipo de licença. Segundo a desenvolvedora Nora Tindall, que obteve a informação e a publicou em seu perfil no Twitter, não há precedentes para a violação de conteúdos protegidos por direitos autorais no treinamento de uma IA e este seria um momento decisivo para o futuro do software livre em geral.
